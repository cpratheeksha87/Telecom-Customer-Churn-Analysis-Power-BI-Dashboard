# 📊 Telecom Customer Churn Analysis — Power BI Dashboard

An end-to-end Power BI report analyzing customer churn behavior across demographic, contractual, usage, and financial dimensions for a telecom dataset. Built as a multi-page interactive dashboard with 10 report pages and 40+ visuals.

---

## 📁 Repository Contents

| File | Description |
|------|-------------|
| `3_5_more_insights_solution.pbix` | Power BI Desktop report file (source + data model) |

---

## 📋 Report Pages Overview

| Page | Key Visuals | Focus |
|------|-------------|-------|
| **Overview** | KPI cards, bar chart, donut chart | High-level churn summary |
| **Churn Demographics** | Table, bar chart, combo chart, KPI cards | Who is churning by demographic |
| **Groups and Categories** | Combo chart, pie, column, donut, multi-row card | Churn by customer segments |
| **Age Groups** | Combo charts, slicer | Churn trends across age bands |
| **Unlimited Plan** | Table, bar chart | Churn among unlimited plan subscribers |
| **International Calls** | Pivot table, map visual | Geographic and usage-based churn |
| **Contract Type** | Pie chart, line charts | Churn by contract length |
| **Payment and Contract** | KPI cards, scatter chart, slicer | Payment method vs. churn correlation |
| **Extra Charges** | Bar chart, KPI cards | Impact of extra charges on churn |
| **Insights** | KPI cards, line chart | Summary insights and trends |

---

## 🛠️ Built With

- **Power BI Desktop** (Version 2.91.884.0, released September 2022)
- **DAX** for calculated measures and KPIs
- **Power Query (M)** for data transformation
- Default theme: CY21SU02

---

## 📌 Key Metrics Analyzed

- Overall churn rate and churn count
- Churn by age group, contract type, and payment method
- Impact of international plan and extra charges on churn likelihood
- Geographic distribution of churned customers
- Group/category segmentation and churn patterns

---

## 🚀 Getting Started

### Prerequisites
- [Power BI Desktop](https://powerbi.microsoft.com/desktop/) (free download)

### Steps
1. Clone or download this repository
2. Open `3_5_more_insights_solution.pbix` in Power BI Desktop
3. Navigate through the report pages using the tabs at the bottom
4. Use slicers and filters to interact with the data
